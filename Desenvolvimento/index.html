Instruções de Desenvolvimento

1) Linux

É bom ter o pyenv instalando para configurar as versões do python.

1.1) Instalando o ambiente python para o desenvolvimento

Caso você queria contribuir para o projeto os comandos para a instalação do ambiente de desenvolvimento são:

1
2
3
4
python -m venv .venv --upgrade-deps
source .venv/bin/activate
pip install pip-tools
pip install -e ".[dev]"

Parar verificar a instalação:

1
pip show confiacim

1.2) Rodando os testes

Par rodar os testes basta:

1
pytest

Para mostar as cobertura de teste

1
pytest --cov=src/confiacim

1.4) Gerenciamento dos pacotes python

Checar vunerabilidades:

1
pip-audit

Para checar updates:

1
pip list --outdated

Para atualizar todos os pacotes:

1
2
3
4
pip-compile requirements/minimal.in -U
pip-compile requirements/plot.in -U
pip-compile requirements/dev.in -U
pip-compile requirements/doc.in -U

1.5) Makefile (depreciado)

No makefile existe algums comando uteis para o desenvolvimento. Para lista-los basta:

1
make help

2) Windows

Para ter ferramentas como make pode-se usar o MSYS2.

2.1) Powershell ou CMD

Instalando o projeto:

1
2
3
4
python -m venv .venv --upgrade-deps
.venv\Scripts\activate
pip install pip-tools
pip install -e ".[dev]"

Rodando os testes:

1
pytest

2.2) WSL

Instalando o projeto:

1
2
3
4
python -m venv .venv --upgrade-deps
source .venv/bin/activate
pip install pip-tools
pip install -e ".[dev]"

Rondando os testes:

1
pytest

3) Precommit

Instalando precommit

1
pre-commit install

4) Ferramentas de desvolvimento

Foi usado taskipy para facilitar o uso das ferramentas. Para listar os comandos disponíveis:

1
task -l

Por exemplo para rodar o black e isort:

1
task fmt
