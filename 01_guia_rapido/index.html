Guia rápido

Necessário ter pelo menos o python 3.11.

Warning

No pacote do confiacim são disponibilizadas versão do tencim1D com a abertura de interface de 1.0e-08.

1) Instalação

O confiacim foi empacotado utlizando o setuptools. Ele pode ser usando localmente de forma standalone como uma aplicação CLI ou importado por outras aplicaççoes. Nesse cenário indica-se instala-lo direto do pacote *.whl. O pacote python .whl pode ser obtido aqui. Para instalar essa versão básica basta:

1
2
3
4
5
```bash
python -m venv .venv --upgrade-deps
source .venv/bin/activate
pip install confiacim-x.y.z-py3-none-any.whl
```

Info

Após feita a instalação só é preciso ativar o ambiente virtual com:

1
source .venv/bin/activate

Info

No windows para ativar o ambiente virtual basta:

1
.venv\Scripts\activate

Para verificar a instalação basta:

1
2
3
```bash
confiacim --version
```

Para habilitar a funcionalidade de plotar gráficos é preciso instalar as dependencias extras, para tal basta:

1
2
3
4
5
```bash
python -m venv .venv --upgrade-deps
source .venv/bin/activate
pip install confiacim-x.y.z-py3-none-any.whl["plot"]
```

Caso se queria instalar globalmente basta usar o pipx:

1
2
3
```bash
pipx install confiacim-x.y.z-py3-none-any.whl
```

ou não criar um ambiente virtual é instalar no python global:

1
2
3
```bash
pip install confiacim-x.y.z-py3-none-any.whl
```

2) Uso

Inicializando o diretório de simulação:

1
2
3
```bash
confiacim init
```

Na pasta simulation já temos 3 casos simples chamados caso1a, caso1b e case8. Na pasta bin temos o executável do tencim. Esse excutável pode ser subistuido se necessário.

2.1) Análise deterministica

Para rodar a análise deterministica:

1
2
3
```bash
confiacim deterministic run simulation/case1a
```

Para plotar o resultado da simulação deterministica:

1
2
3
```bash
confiacim deterministic plot simulation/case1/output/
```

Para modificar case.dat para ter apenas 20 passos de tempo pode-se usar o new_last_step. Esse comando irá criar uma cópia do case.dat como o nome case_original.dat e depois modificar case.dat:

1
2
3
```bash
confiacim deterministic new_last_step 20 simulation/case1/case.dat
```

Caso seja necessário novamente modificar case.dat pode-se usar o case_original.dat:

1
2
3
```bash
confiacim deterministic new_last_step 30 simulation/case1/case_original.dat
```

2.2) Análise probabilistica com o FORM

Agora para rodar a análise probabilistica usando o método FORM basta:

1
2
3
```bash
confiacim form run -s simulation/case1
```

Os resultados são salvos na pasta simulation/case1/output/form. Com a uso da opção -s pode-se plotar os curvar de RCs. Para vizualizar todos os resultados:

1
2
3
```bash
confiacim form plot all simulation/case1a/output/
```

Vizualizando os resultados da iteração 3:

1
2
3
```bash
confiacim form plot it 3 simulation/case1a/output/
```

Caso você queria só os arquivos básicos da solução basta não usar o -s:

1
2
3
```bash
confiacim form run simulation/case1
```

2.3) Análise probablística com Monte Carlo

Para rodar a análise Monte Carlo basta fazer:

1
2
3
```bash
confiacim mc run simulation/case1a/
```

Para apenas gerar as amostras:

1
2
3
```bash
confiacim mc mk_samples run simulation/case1a/
```

Para plotar o histograma das amostras de uma distribuição:

1
2
3
```bash
confiacim mc hist E_c simulation/case1
```

2.4) Vizualizando as distribuições

Vizualizando a curva pdf da variável E_c:

1
2
3
```bash
confiacim dist pdf E_c simulation/case1a
```

Vizualizando a curva cdf da variável E_c:

1
2
3
```bash
confiacim dist cdf E_c simulation/case1a
```

Vizualizando a curva ppf da variável E_c:

1
2
3
```bash
confiacim dist ppf E_c simulation/case1a
```

2.5) Rodando uma análise do tencim1D de forma direta

Para rodar um análise pura do o tencim1D pode-se usar o comando

1
2
3
```bash
confiacim tencim run simulation/pos-sal-top-thick
```

Este comando irá colocar os resultados na pasta output.
