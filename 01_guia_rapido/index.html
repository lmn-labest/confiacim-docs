Guia rápido

Necessário ter pelo menos o python 3.10.

Danger

O tencim1D utilizado tem o valor de abertura da interface de 1e-08.

1) Instalação

Para instalar essa versão básica basta utilizar o .whl:

1
2
3
python -m venv .venv --upgrade-deps
source .venv/bin/activate
pip install confiacim-x.y.z-py3-none-any.whl["plot"]

Para verificar a instalação basta:

1
confiacim --version

2) Uso

Inicializando o diretório de simulação

1
confiacim init

Na pasta simulation já temos um caso simples chamado caso1a e na pasta bin o executável do tencim. Esse excutável pode ser subistuido se necessário.

2.1) Análise deterministica

Para roda a análise deterministica:

1
confiacim deterministic run simulation/case1a simulation/case1a/output/run1

Plotando o resultado da simulação deterministica:

1
confiacim deterministic plot simulation/case1/output/run1

Para modificar case.dat para ter apenas 20 passos de tempo pode-se usar o deterministic. Esse comando irá criar uma cópia do case.dat como o nome case_original.dat e depois modificar case.dat:

1
confiacim deterministic new_last_step 20 simulation/case1/case.dat

Caso seja necessario novamente modificar case.dat pode-se usar o case_original.dat:

1
confiacim deterministic new_last_step 30 simulation/case1/case_original.dat

2.1) Análise determistica com o form

Agora para rodar a análise Para rodar o método form:

1
confiacim form run simulation/case1 simulation/case1/output/run1

Vizualizando todos os resultados:

1
confiacim form plot all simulation/case1a/run1

Vizualizando os resultados da iteração 3:

1
confiacim form plot it 3 simulation/case1/output/run1
