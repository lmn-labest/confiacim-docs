Templates das variáveis aleatórias

Para gerar os arquivos de entrada dinamicamente foi usado o jinja2. Os arquivos serão procurados em uma subpasta templates que esta no diretório dos arquivos de entrada .dat. A estrutura de pasta será similar à:

│   ├── initialtemperature.dat
│   ├── loads.dat
│   ├── materials.dat
│   ├── mesh.dat
│   ├── setpnode.dat
│   └── setptime.dat
│   └── templates
│       ├── hidrationprop.jinja
│       └── materials.jinja

Note

O jinja2 possui vários filtros uteis como pode se ver na documentação. Nós utilizamos principalmente o filtro format.

1) Materials

Para gerar o aquivo materials.dat basta criar um arquivo materials.jinja e coloca-lo na pasta templates.

  4 1 {{ "%.15e"|format(E_f * 3.00e10) }} {{ "%.15e"|format(poisson_f * .15) }} {{ "%.15e"|format(thermal_expansion_f * 3.900e-05) }} 0 0 {{ "%.15e"|format(thermal_conductivity_f * 2.300e+00) }} {{ "%.15e"|format(volumetric_heat_capacity_f * 4.032e+06) }} 0 0 0
end materials
return

3) Initialtemperature

Para gerar o aquivo initialtemperature.dat basta criar um arquivo initicaltemperature.jinja e coloca-lo na pasta templates. O numero 5 deve ser substituido pelo pelo número por nnode + 1.

end initialtemp
